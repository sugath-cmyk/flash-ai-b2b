Usage Guidelines

  • Primary Logo: Use Concept 1 for most applications
  • Favicon: Use the 16x16 or 32x32 icon versions
  • App Icon: Use the 64x64 version with rounded corners
  • Minimum Size: Don't use the full logo smaller than 120px wide
  • Clear Space: Maintain padding equal to the icon height around the logo
  • Dark Backgrounds: Switch to the light purple (#A78BFA) variant
